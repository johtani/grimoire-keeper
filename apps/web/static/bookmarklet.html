ブックマークレット設定

ブラウザで見ているページを1クリックで Slack 経由 Grimoire Keeper に登録できます。

クリックすると、現在のURLと任意のメモが @Bot URL 形式でクリップボードにコピーされ、 Slack デスクトップアプリの指定チャンネルが開きます。あとは貼り付けて送信するだけです。

1 Slack の ID を入力
調べ方: Slack Web (app.slack.com) を開き、URL の https://app.slack.com/client/T〇〇〇/... の T で始まる部分をコピー
調べ方: Slack でチャンネル名を右クリック →「リンクをコピー」→ コピーした URL 末尾の C で始まる部分をコピー
調べ方: Slack でボットのプロフィールを開く → 「…」メニュー →「メンバー ID をコピー」
2 ブックマークバーに追加

ブックマークバーが表示されていない場合は Ctrl+Shift+B (Windows/Linux) / Cmd+Shift+B (Mac) で表示できます。

下のボタンをブックマークバーにドラッグしてください

ID をすべて入力するとリンクが生成されます 📚 Grimoire に登録

ドラッグが難しい場合は、リンクを右クリック →「ブックマークに追加」でも登録できます。

3 使い方
  1. 登録したいページを開く
  2. ブックマークバーの 「📚 Grimoire に登録」 をクリック
  3. メモを入力（任意）して OK
  4. Slack デスクトップアプリが開く
  5. Cmd+V / Ctrl+V で貼り付けて Enter で送信

ID をすべて入力するとコードが表示されます。
